# 3D Tetris
A 3D Tetris-style game built in Unity with C#.

## Features
- **Grid-based movement** — Piece movement, rotation, hard drop, and ghost preview
- **Layer clearing** — Full layers clear with score tracking
- **Dynamic difficulty** — Fall speed increases over time
- **Audio feedback** — Sound effects for piece placement and clearing

## How to Play
Open the project in Unity 2021+ and press Play.

## Controls
| Action | Key |
|---|---|
| Move | Arrow Keys |
| Rotate | Z / X |
| Hard Drop | Space |

## Tech Stack
- **Engine:** Unity
- **Language:** C#
